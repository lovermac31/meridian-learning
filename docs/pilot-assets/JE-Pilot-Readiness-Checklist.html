Scoring Guide For institutional and operator use · not a pass/fail test
18–20
High Readiness
Proceed to pilot scope conversation. All required items should be checked before launch.
13–17
Moderate Readiness
Identify unchecked items. Address Required gaps before committing. Preferred gaps can follow.
Below 13
Pre-Pilot Advisory
An Audit Sprint or advisory session is recommended before a full pilot is activated.
Operator Note — Internal Use Only
Items marked Required are non-negotiable — a pilot cannot produce reliable evidence without them. Items marked Planning indicate preparation work that should be done before launch but can be addressed during the setup phase. Items marked Preferred improve evidence quality but are not blockers. If all Required items are checked and score is ≥ 13, proceed to scoping. If any Required item is unchecked, address it before advancing.
