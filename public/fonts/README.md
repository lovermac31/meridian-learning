# Self-hosted fonts

## Fraunces

The `fraunces-*.woff2` files are web subsets of Fraunces used by the Jurassic
English homepage hero headline.

- Family: Fraunces
- Designers: Undercase Type, Phaedra Charles, Flavia Zimbardi
- License: SIL Open Font License 1.1
- Upstream: https://github.com/undercasetype/Fraunces
- Google Fonts metadata: https://github.com/google/fonts/tree/main/ofl/fraunces
- Upstream source commit recorded by Google Fonts: `d6d385783609ceb11ac0f220f3abd9f1631c8a36`
- Included subsets: Latin, Latin Extended, Vietnamese
- Included instance: weight 600 at optical size 144

The complete license and copyright notice are in `Fraunces-OFL.txt` and must
remain distributed with the font files.
